STEAM GAMES INTERNET ACCESS GETTING HELP
We operate a steam cache server at the LAN, so most steam games should download very quickly. This includes popular games we play here, such as Team Fortress 2. We have limited internet capacity shared between alot of people. We ask that you try not to download anything large from the internet, or stream Youtube/Twitch/etc. while here.

This does not apply to steam games, however, which are provided from the steam cache server.

Check our download repository for some popular files, such as graphics drivers.
